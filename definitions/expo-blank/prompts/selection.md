# Template Selection

Modern React Native + Expo mobile app starter

Use when:
- Mobile-first apps (iOS and Android)
- Client-side apps with local storage
- Dashboards, utilities, productivity apps
- Simple, fast Expo projects

Avoid when:
- Web-only applications needed
- Complex backend/server requirements
- Heavy real-time data syncing

Built with:
- Expo SDK 52, expo-router, React Native
- Pre-built UI components (Button, Card, TextInput)
- @expo/vector-icons, react-native-reanimated
- AsyncStorage for local persistence
- TypeScript, ESLint
